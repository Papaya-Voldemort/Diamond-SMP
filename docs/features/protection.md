# Protection

Diamond SMP does not implement land claims, but it still enforces several protective rules.

## Combat and Interaction Protections

- trust-hit prevents friendly damage between trusted players
- friendly fire is disabled inside party PvP teams
- reward villagers are invulnerable and owner-bound
- non-owners cannot use another player's reward villager unless they have admin permission
- active reward villagers can be despawned cleanly by admins

## Progression Protections

- blocked crafting, smithing, enchanting, anvil, and trade outputs stop illegal endgame creation
- loot cleanup prevents backdoor access
- End access can be hard-closed and players are ejected to the Overworld if needed

## World-Scale Admin Controls

- world border management via `/dsborder`
- server-wide rules GUI via `/rules`
