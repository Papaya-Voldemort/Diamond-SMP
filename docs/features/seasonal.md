# Seasonal Systems

Diamond SMP does not currently ship an automatic seasonal scheduler, battle pass, or rotating content calendar.

## What You Can Still Run Seasonally

- rotate `rules.yml`
- adjust villager prices and stock
- change allowed PvP kits and modes
- run manual server events on a schedule
- reset purchase history between seasons
- reopen or close the End when your season plan requires it

## Practical Seasonal Reset Checklist

1. `/godvillager reset`
2. review `villagers.yml`
3. review `config.yml` PvP and world values
4. update `rules.yml`
5. announce the new event cadence to players
