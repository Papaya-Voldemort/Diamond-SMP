# Admin and Integrations

## Admin Systems

- reward villager spawning, clearing, and purchase resets
- event start, stop, and status control
- world border control
- End access control
- god-item validation kit

## Operator Responsibilities

- decide when events happen
- decide which villager reward bucket is attached to each event
- tune prices and cooldowns
- decide whether kits are public or admin-only
- reset retired villager stock between seasons if desired

## PlaceholderAPI

If PlaceholderAPI is installed, Diamond SMP registers a persistent expansion exposing event information.

## What Is Not Implemented

- no Vault hooks
- no database-backed admin panel
- no web dashboard
- no LuckPerms-specific integration layer
