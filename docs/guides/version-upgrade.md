# Version Upgrade Guide

Current target:

- Minecraft Java Edition `1.21.11`
- Paper API `1.21.11-R0.1-SNAPSHOT`

When upgrading:

1. Confirm official Minecraft release target
2. Update `gradle/libs.versions.toml`
3. Update `plugin.yml` api-version
4. Review Paper API breakage
5. Run startup smoke tests on the new server version
