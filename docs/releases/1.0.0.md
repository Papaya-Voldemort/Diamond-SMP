# Diamond SMP 1.0.0

Diamond SMP `1.0.0` is the first release of the plugin as a standalone Paper project for Minecraft `1.21.11`.

## Highlights

- Event-driven endgame progression with Name Tag and Cat Hunt server events
- Reward villagers that bind powerful item access to event winners
- Global retirement of purchased villager stock until admin reset
- God items with custom combat, mining, mobility, and survival effects
- Party PvP with kits, arena flow, rematch support, and full state restoration
- Utility command suite for teleport requests, trust, rules, spawn, kits, and event status
- Admin controls for event flow, world border management, villager management, and End access
- PlaceholderAPI support for event placeholders

## Runtime

- Minecraft `1.21.11`
- Paper `1.21.11`
- Java `21`

## Artifact

- `plugin-bootstrap/build/libs/Diamond-SMP-1.0.0.jar`

## Validation

- `./gradlew build`
- Verified successful shadow jar build through `:plugin-bootstrap:shadowJar`
