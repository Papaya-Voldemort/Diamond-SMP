# Diamond SMP 1.0.2

Diamond SMP `1.0.2` completes the branded server bootstrap by downloading the missing companion plugins from the web instead of only seeding their configs.

## Highlights

- Automatically downloads PlaceholderAPI `2.12.2`, TAB `5.5.0`, and Custom Join Messages `17.9.1` when those jars are missing
- Uses exact Modrinth CDN URLs exposed in `config.yml -> branding.companion-downloads`
- Seeds PlaceholderAPI, TAB, and CustomJoinMessages configs with the same branded files used on the Diamond SMP test server
- Keeps party PvP opt-in and disabled by default until `config.yml -> pvp.enabled` is turned on

## Runtime

- Minecraft `1.21.11`
- Paper `1.21.11`
- Java `21`

## Artifact

- `plugin-bootstrap/build/libs/Diamond-SMP-1.0.2.jar`

## Validation

- `./gradlew clean build`
- Verified companion dependency defaults point at exact Modrinth jar URLs
- Verified branding defaults still match the Diamond SMP test server identity
