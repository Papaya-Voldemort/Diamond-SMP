# Diamond SMP 1.0.1

Diamond SMP `1.0.1` refines the first release with a safer PvP rollout and a bundled branding sync based on the Diamond SMP test server.

## Highlights

- Party PvP beta is now opt-in and disabled by default until `config.yml -> pvp.enabled` is turned on
- Server branding sync now applies the Diamond SMP test server MOTD and server icon
- Bundled companion configs now seed TAB and CustomJoinMessages with the same branded files used on the test server when those plugin folders already exist
- Default `messages.yml`, `rules.yml`, and `events.yml` now match the exact branded copy from the test server

## Runtime

- Minecraft `1.21.11`
- Paper `1.21.11`
- Java `21`

## Artifact

- `plugin-bootstrap/build/libs/Diamond-SMP-1.0.1.jar`

## Validation

- `./gradlew clean build`
- Verified branded companion resources package into the release jar
- Verified PvP defaults remain beta-only and disabled until explicitly enabled
