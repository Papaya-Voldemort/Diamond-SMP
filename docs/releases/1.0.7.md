# Diamond SMP 1.0.7

Diamond SMP `1.0.7` adds the full God Beacon prestige path as a persistent, local-only infrastructure progression system for base building and diamond sink gameplay.

## Highlights

- Adds Infused, Resonant, Ascended, and God Beacon tiers on top of vanilla beacons
- Persists tier and selected effects per placed beacon, including break-to-item portability and restore-on-place behavior
- Adds crafted Core catalysts plus upgrade-time payment from player inventory and nearby base storage
- Adds `/beacon` inspection, stats, core distribution, tier mutation, and debug tooling
- Adds Tier 3 and Tier 4 passive systems including linger, regeneration throughput bonuses, hunger efficiency, durability efficiency, mining-speed bonus, visuals, sounds, and smelting aura
